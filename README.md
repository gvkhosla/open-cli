# Open CLI

Open CLI is a curated directory of command-line tools.

It helps people:
- discover useful CLIs
- compare popularity and momentum
- copy install commands
- understand what to run first

## Stack
- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui

## Local development

```bash
npm install
npm run dev
```

## Production
Deployed on Vercel.
